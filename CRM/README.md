▼概要<br>
メルマガ配信データを元に、開封の有無等を集計<br>
<br>
▼使用言語<br>
Python<br>
<br>
▼目的<br>
メルマガ配信データ集計作業時間の大幅短縮<br>
<br>
▼背景<br>
分析担当者から、顧客の購入回数等のデータから顧客をランク分けする分析手法は無いかと相談を受け、<br>
CPM分析を行うスプレッドシートとLooker Studioを用いたデータを作成。<br>
<br>
基本的には元となる顧客情報のcsvデータを変更すれば、分析データを更新可能ではあるが、<br>
更新作業時の作業内容が少々複雑であることと、データが重いことにより出力までに時間が掛かってしまうことが問題点。<br>
<br>
そのため、顧客情報のcsvデータ1つを落として指定のディレクトリに置くだけで、<br>
最新のCPM分析のデータを出力できるようにPythonプログラムを作成。
